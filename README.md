**Глобальное улучшение UI/UX дизайна и функционала для донат-магазинов GameStores.**

**GameStores Styles** — это стиль, превращающий стандартный стиль магазина в современный, отзывчивый и уникальный интерфейс. Проект полностью Open Source.

![Main Interface](https://i.postimg.cc/GpCRbDFt/1111111.png)

## ✨ Особенности

* Полная переработка визуального стиля (цвета, отступы, шрифты).
* Готовые HTML-шаблоны для виджетов (мониторинг, новости, профиль).
* JS-модули для слайдеров, скрытия лишних элементов и динамического контента.
* Все настройки вынесены в начало файла — не нужно быть программистом, чтобы включить нужные функции.
* Красивое оформление описания товаров (карточки предметов).

## 📦 Установка

### Шаг 1: Подготовка магазина
Зайдите в панель управления магазином -> **Дизайн**.
> ⚠️ **Важно:** Убедитесь, что выбран стиль шаблона **"Red Black"**. На других стилях корректная работа не гарантируется.

### Шаг 2: Подключение кода
Перейдите в **Внешний вид** -> **Продвинутая конфигурация**:
1.  Скопируйте содержимое файла `css.css` из репозитория и вставьте в поле **CSS**.
2.  Скопируйте содержимое файла `js.js` и вставьте в поле **JS**.

### Шаг 3: Добавление виджетов
Перейдите в **Внешний вид** -> **Виджеты**:
1.  Нажмите "Добавить виджет" -> **Пользовательский виджет**.
2.  Откройте папку `Widget` в этом репозитории.
3.  Скопируйте HTML-код нужного виджета и вставьте в поле для кода.
4.  Замените текст и ссылки на свои.

### Шаг 4: Оформление товаров
Перейдите в **Управление** -> **Товары**:
1.  При создании или редактировании товара пролистайте до поля "Описание".
2.  Включите режим кода (кнопка `</>`).
3.  Вставьте HTML-шаблон из папки `Products`.

## ⚙️ Настройка (JS)

В файле `js.js` в самом начале есть блок настроек. Вы можете менять их `true` (включить) или `false` (выключить):

```javascript
// --- Конфигурация ---

this.infoForIP = true;               // Показать блок с юридической информацией ИП
this.swiper = true;                  // Включить анимированный слайдер в шапке
this.hideProductName = true;         // Скрыть всплывающие названия товаров (для чистоты UI)
this.hideLanguageSwitcher = true;    // Скрыть переключатель языков
this.bonuses = true;                 // Показывать блок с бонусами при пополнении
```

## 🎨 Для продвинутых (CSS и JS)

Вы можете изменить цветовую схему магазина за пару секунд. Откройте css.css и найдите блок :root в начале файла. Просто поменяйте HEX-коды цветов на свои. Улучшайте и доробатывайте JS как вам угодно.
